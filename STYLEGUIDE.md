# Frontend Styleguide

This document defines the design standards for our frontend.  
It ensures a consistent user experience across all components.

---

## Icons
To establish a unified design language, all icons must come from
[Google Material Symbols](https://fonts.google.com/icons)
with the **Rounded** and **Material Symbols (New)** styles enabled.

### Usage
- Use **SVG** format whenever possible.
- Default size: **24 Ã— 24 px**.
- Icons inherit text color unless explicitly styled.
- Avoid stretching, rotating, or recoloring icons outside the design system.
